Data science and machine learning resources for screen reader users

Project

In recent years, technologies such as data science and machine learning have been adopted in various fields. In the future, there will be more and more opportunities to learn them from an early stage in various types of school education. On the other hand, in order to acquire and utilize such technologies, it is essential to analyze huge amounts of data using graphs and other tools.

In this project, I would like to develop the necessary information and tools for screen reader users to learn these technologies by relying on auditory information rather than visual information, in preparation for the future. At present, I have developed two libraries: a library that notifies users of visual status changes on google colaboratory with sound, and an alternative library for plots that allows users to check graph data with sound.

The objective is for screen reader users to be able to run through and understand some typical machine learning tutorials. Eventually, I also hope that people who cannot see or have difficulty seeing will be able to use the tools for other scientific research, and that they will be able to find a high-paying job such as data scientist.

Libraries

audio-plot-lib

This library provides graph sonification functions in Google Colab

The following is an example of a voiced graph. Two graphs, sine wave and cosine wave, are sonified by this library. First, click on the next button to activate the audio. Note that this can be omitted in Colab.

After pressing the button, next move the mouse cursor appropriately to find if the sound "Enter graph" is played out. Then, while you are on the graph image, you can check the values plotted on the graph as pitches by moving the mouse left or right.

In the initial state, you can hear the sine wave values. By double-clicking, you can now switch to the cosine value. You can also single-click to read out the x and y values for that point.

colab-a11y-utils

This library provides the following functions in Google Colab

  • One beep sound when executed
  • Two beep sounds when normal completion
  • Two buzzer sounds when abnormal completion
  • Sound notification of progress bar by tqdm
  • Simple audio output function that can be used instead of print
